Neovim + Tmux: My Ubuntu Terminal Dev Environment

Published on February 18, 2025

Over the last year, I've refined my terminal development environment to be as efficient and lightweight as possible. My current setup revolves around Neovim, Tmux, and Ubuntu. Here's a quick overview of how I configure my tools for maximum productivity:

Neovim Configuration

Neovim is my editor of choice due to its speed and extensibility. Below is a snippet of my configuration:

-- This file can be loaded by calling `lua require('plugins')` from your init.vim

-- Only required if you have packer configured as `opt`
vim.cmd [[packadd packer.nvim]]

return require('packer').startup(function(use)
  -- Packer can manage itself
  use 'wbthomason/packer.nvim'

  use {
    'nvim-telescope/telescope.nvim', tag = '0.1.8',
-- or                            , branch = '0.1.x',
    requires = { {'nvim-lua/plenary.nvim'} }
  }

  use({ 
	"rose-pine/neovim", 
	name = "rose-pine",
	config = function()
		vim.cmd("colorscheme rose-pine")
	end
  })
  use('nvim-treesitter/nvim-treesitter', {run = ':TSUpdate'})
  use('nvim-treesitter/playground')
  use('theprimeagen/harpoon')
  use('mbbill/undotree')
  use('tpope/vim-fugitive')

  use {
	'VonHeikemen/lsp-zero.nvim',
	requires = {
		-- LSP Support
		{'neovim/nvim-lspconfig'},
		{'williamboman/mason.nvim'},
		{'williamboman/mason-lspconfig.nvim'},

		-- Autocompletion
		{'hrsh7th/nvim-cmp'},
		{'hrsh7th/cmp-buffer'},
		{'hrsh7th/cmp-path'},
		{'saadparwaiz1/cmp_luasnip'},
		{'hrsh7th/cmp-nvim-lsp'},
		{'hrsh7th/cmp-nvim-lua'},

		-- Snippets
		{'L3MON4D3/LuaSnip'},
		{'rafamadriz/friendly-snippets'},
	}
  }

end)

Tmux for Multiplexing

Tmux allows me to manage multiple terminal sessions efficiently. I use a minimal configuration with custom keybindings to split panes and navigate between them seamlessly.

# vi mode
set-window-option -g mode-keys vi
set -g default-terminal "screen-256color"
set -g default-shell "$SHELL"

# remaps
unbind C-b
set -g prefix C-s
bind-key C-s send-prefix

Why This Setup?

This combination of tools provides a highly customizable and efficient workflow. Neovim's extensibility, combined with Tmux's session management, allows me to focus entirely on coding without distractions.

If you're looking to streamline your terminal-based development workflow, I highly recommend giving this setup a try!
